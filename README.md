# OPERATION SANDSTORM

three.js製のブラウザFPSゲーム。

## 遊び方

- プレイ: https://hayashiii-ghub.github.io/karakin-demo/
- ローカルで開く場合は `index.html` をダブルクリックでも開けます（three.jsはローカル同梱済み・オフライン可）。
- うまく動かない場合は `python3 -m http.server` で `localhost:8000` にアクセスしてください。

## リアル志向のポイント

### ガンプレイ

- フルオート750rpm、リコイルパターン（垂直キック＋ランダム横ブレ＋回復）、連射で拡がる集弾率
- 右クリックADS：FOVズーム＋レッドドットサイト照準＋感度低下＋呼吸ブレ
- 30発マガジン＋予備弾薬、リロード2.1秒、薬莢排出、マズルフラッシュ＋閃光、トレーサー
- 部位判定：ヘッドショット一撃（+150）/ 胴3発 / 四肢減衰

### 敵AI

- パトロール → 視認 or 銃声を聞きつけて索敵（距離で位置推定誤差）→ 交戦
- 反応遅延、バースト射撃、左右ストレイフ、しゃがみ、間合い管理、距離減衰命中率
- 撃たれると赤い方向インジケータ＋ヴィネット、近くを弾が通ると「crack」音

### その他

- 全音響をWeb Audioで手続き合成（銃声・足音・風・遠雷）
- HP自動回復、敵ドロップ（弾薬/救急キット）、キルフィード、死亡カメラ、統計付きリザルト

ヘッドレスChromiumで実戦シミュレーション検証済み（射撃・キル・ウェーブ・死亡までエラー0確認）。`?debug=1&shoot=1&fast=150` をURLに付けると自動戦闘モードで見られます。
